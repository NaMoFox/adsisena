En HTML5 no es necesario proporcionar valores para todos los atributos, cómo los controles, reproducción automática y atributos de bucle utilizados con el elemento <video< Estos atributos son como interruptores de encendido/apagado. Si el atributo está presente, activa esa opción. Si se emite el atributo la opción está desactivada. Si el navegador no admite el elemento <video< o el formato de vídeo utilizado, se mostrará lo que entre la <video< de apertura y las etiquetas de <video< de cierre

preload este atributo le indica al navegador que hacer cuando se carga la página. Puede tener uno de tres valores:
none: el navegador no debe cargar el vídeo hasta que el usuario presiona Play.
metadata:

el navegador debe simplemente recopilar informacion como el tamaño, el primer fotograma, la lista de pistas y la duración.
