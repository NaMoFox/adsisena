<audio> HTML5 introdujo el elemento <audio> para incluir archivos de audio en sus paginas. Al igual que con el video HTML5, los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un numero de atributos que le permiten controlar la reproduccion de audio.

src: este atributo especifica la rura de acceso al archivo de audio.

controls: este atributo indica si el reproductor debe mostrar los controles. si no utiliza este atributo, no se mostrará controles de forma predeterminada. También puede especificar sus propios controles mediante JavaScript.

autoplay: la presencia de este atributo indica que el audio debe empezar a reproducirse automáticamente. (se considera una mejor práctica permitir que Los visitantes elijan Reproducir audio.)

p>

loop: este tributo específica que la pista de audio debe reproducirse de nuevo una vez que haya finalizado
